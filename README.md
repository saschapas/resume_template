# Résumé Template
This is a template for a résumé written in LaTex. 

I wrote this for one for my own job applications (actually the first computer science job I applied for).

As it is just a template, the spaces have to be filled, as well as potential points added (marked by '*EXAMPLE*').

For it to compile LaTex has to be installed on the system it is written on and some kind of IDE is needed to edit it.

I used VSCode in combination with the extension [LaTex Workshop by James Yu](https://github.com/James-Yu/LaTeX-Workshop "GitHub page of the extension"). Sadly it comes a totally annoying auto-somplete AI which needs to be turned off.

Formatting options are mostly commented, further information can be acquired in the documentation of these.

These are the packages used:

- [titlesec](https://ctan.org/pkg/titlesec "ctan titlesec page")
- [titling](https://texdoc.org/serve/titling.pdf/0 "titling package pdf")
- [geometry](https://ctan.org/pkg/geometry "ctan geometry page")
- [scrextend](https://ctan.org/pkg/scrextend "ctan scrextend page")
- [needspace](https://ctan.org/pkg/needspace "ctan needspace page")
- [hyperref](https://ctan.org/pkg/hyperref "ctan hyperref page")
